# Vue Tabs

Vue Tabs is a tab system written for Vue.js

## Table of Contents

- [Installing](#installing)
- [Options](#options)
- [Todo](#todo)

## Installing

`npm install @mmano/vue-tabs`

```javascript
import Vue from 'vue';
import Tabs from '@mmano/vue-tabs';
Vue.use(Tabs);

new Vue({
  el: '#app',
});
```

```html
<vue-tabs>
  <vue-tab name="Tab 1">
    <p>Content for tab 1</p>
  </vue-tab>
  <vue-tab name="Tab 2">
    <p>Content for tab 2</p>
  </vue-tab>
</vue-tabs>
```
